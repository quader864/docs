## Response

<ResponseField name="message" type="string">
  Logout confirmation message
</ResponseField>

<RequestExample>
```bash cURL
curl -X POST https://api-verifier-2.onrender.com/logout
```
</RequestExample>

<ResponseExample>
```json 200
{
  "message": "Logged out"
}
```
</ResponseExample>
